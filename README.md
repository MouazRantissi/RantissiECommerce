### Customer Features
* **Authentication:** Secure user login and registration.
* **Dynamic Browsing:** Filter products by specific categories (Menswear, Womenswear, Footwear, etc.).
* **Cart Management:** Add products to the cart, dynamic total price calculation, and the ability to remove items.
* **Seamless Checkout:** Push orders directly to the cloud and instantly update delivery status.
* **Order History:** View past purchases with timestamped logs and total item counts.

### Admin Dashboard (Role-Based Access)
* **Inventory Management:** Full CRUD (Create, Read, Update, Delete) capabilities for Products and Categories.
* **SuperAdmin Protection:** Strict gatekeeper logic preventing standard admins from modifying protected user accounts.
* **Recycle Bin System:** A dedicated trash node allowing admins to safely restore accidentally deleted items or permanently wipe them from the database.
* **Image Handling:** Dynamic product image rendering via URL integration.
